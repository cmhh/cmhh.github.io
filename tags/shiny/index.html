Using R Packages to Disseminate Data

Published at July 20, 2016 ·  10 min read

Introduction R packages can include datasets, and it is very common to do so. This shouldn’t really come as a surprise–R is a statistical language in the end, and it is normal to include working examples with pacakges; and including datasets against which to run unit tests and other examples is clearly a useful thing to do. But datasets themselves aren’t central to most packages–statistical methodologies often are, and the data is there for testing or illustrative purposes....
